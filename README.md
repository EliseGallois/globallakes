# Summary

This repository contains resources and tools for global lakes research. The aim is to provide datasets, models, and visualizations to support environmental and ecological analyses.

# User Manual

1. **Installation**
   - Install the required packages:
     ```R
     install.packages(c('ggplot2', 'dplyr', 'leaflet'))
     ```

2. **Loading Data**
   - Load the data using the provided functions in the package.

3. **Running Analyses**
   - Use the functions in the package to perform analyses and generate visualizations.
